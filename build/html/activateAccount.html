Activate Account¶

This function is for the formal acceptance and activation of user to create an account to iNoCulator. To access this feature, user must click the ‘link’ from the Invitation email and it will open a tab which will redirect to this screen:

Limitations:

  • Upper case only on the first letters. (first name and last name)

  • Use only alphanumeric keys including ñ and Ñ, spaces and no special characters.

After completing the form, click ‘Activate’ button to proceed with creation of account. User will receive a confirmation email informing that account is already created.
